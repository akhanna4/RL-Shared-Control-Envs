# LunarLander-v2 (Continuous)
Keyboard code for the LunarLander environment with a continuous action space.

## Installation

The following dependencies need to be installed:

```
pip install gym
pip install Box2D
pip install torch torchvision
pip install ipython
pip install matplotlib
pip install nose
```

## Run 

To run the program:

```
python main.py
```
